# Эхо-Архипелаг

Доступная браузерная кооперативная аудио-игра про спасение на лодке.

## Что работает

- Управление лодкой с инерцией в тихой речной бухте без постоянного течения.
- Отдельный режим VoiceOver без двойных команд и без скачущего фокуса.
- Пространственный сонар, двигатель, перегрев, повреждения, течь, насос и спасательный трос.
- Одиночная игра с помощником и кооператив капитан/оператор.
- Раздельные локальные записи спокойной речной воды, кильватера и воды внутри корпуса без сетевой зависимости во время игры.
- Вода в корпусе постепенно закрывает общий low-pass-фильтр: меняется тембр всего игрового микса, а не только громкость воды.
- При полном затоплении или нулевой прочности корпуса остаётся 45 секунд: вода должна быть не выше 35%, корпус — не ниже 5%. Игра просит только реально нужное действие, а штормовой таймер на время аварии ставится на паузу.
- Центральный сектор стереомаяка полностью очищен от обломков; твёрдое препятствие останавливает удерживаемый газ после первого контакта и не пропускает лодку насквозь.
- На мобильном устройстве потеря `pointerup` больше не оставляет управление зажатым, а одно нажатие обычного тормоза в режиме экранного диктора доводит лодку до остановки.
- В обычном режиме сонар и стереомаяк всегда указывают прямо на текущего человека или гавань; скрытых координатных переключений и промежуточных точек нет.
- В зоне 14 метров маяк замолкает, трос одним нажатием фиксирует и останавливает лодку, а финальный створ гавани автоматически ограничивает опасную скорость.
- В дружелюбном режиме сонар ведёт прямо к самой цели без промежуточных точек. После центрирования включается удержание курса, которое убирает остаточный руль и автоматически исправляет микроскопический уход.
- На первом уровне обязательные прямые линии очищены от препятствий; на следующих операциях обломки образуют слышимые проходы, не перекрывая точную центральную линию маяка.
- Со второго уровня рядом с сонаром открывается явный переключатель обычного и рискованного маршрута. Рискованный режим озвучивает вход и выход узкого прохода, приостанавливает автоподруливание только внутри него и один раз награждает за прохождение без столкновения.
- Одиночная игра теперь состоит из трёх последовательно открываемых операций. Второй уровень открывает магазин спасслужбы, третий — устойчивый катер «Касатка» с отдельным низким тембром двигателя.
- За завершённые операции сохраняются жетоны. В магазине доступны береговой автотормоз на пять секунд, мини-броня и усиленная помпа; покупки постоянны.
- Насос откачивает воду, но не «чинит» течь. Автооткачка одиночного помощника по умолчанию выключена и включается отдельной кнопкой; ручной насос сильнее, а улучшенная помпа добавляет поток поверх выбранного режима.
- На 80% воды моторный отсек заливает и двигатель глохнет. После осушения до 35% и стабилизации корпуса он запускается автоматически за 1,2 секунды либо сразу по газу или кнопке. Четырёхсекундное обслуживание нужно только после настоящего перегрева.
- Заглохший мотор больше не создаёт задний ход, автооткачка помощника не бросает насос во время спасения, а аварийная лодка не может случайно завершить операцию в гавани до стабилизации.
- Незавершённое спасение больше нельзя накапливать короткими нажатиями: после отмены троса частичный прогресс постепенно теряется, а во время полного затопления подача троса блокируется.
- Когда топливо заканчивается, лодка ещё может докатиться до гавани по инерции; после полной остановки вне гавани операция честно завершается вместо бесконечного ложного авторемонта.
- Урон от столкновения растёт нелинейно от скорости: касание на малом ходу остаётся ремонтируемой ошибкой, а удар на полном ходу может немедленно запустить аварийное окно спасения лодки.
- Частые игровые фразы ограничены 105 знаками, статусы и подсказки сокращены; в режиме VoiceOver встроенная речь по умолчанию выключена.
- Локальные записи воды загружаются до первого старта, повторяющаяся аварийная сирена длится всё окно спасения, а после системной речи Web Audio автоматически возобновляется на мобильных устройствах.
- Кнопка «Вернуться к карте операций» сбрасывает таймер, сеть, звук и игровое состояние без перезагрузки страницы.

## Звуки и лицензии

Фоновая речная запись — `Stream` автора mystiscool, Freesound 7138, лицензия CC BY; локальная OGG-копия взята из проекта AmbientSounds. Кильватер и внутренний плеск используют `loop_water_02.ogg` и `loop_water_01.ogg` из CC0 Public Domain Sounds.

Остальные сценарные сигналы и часть механических эффектов используют ранее подключённые CC0/совместимые записи. Сценарные звуки сонара, поворота и спасения не заменялись.

## Локальный запуск

```bash
npm ci
npm test
npm run serve
```

Открой `http://localhost:4173/`.

## Cloudflare Workers Static Assets

```bash
npm ci
npx wrangler login
npm run deploy
```

Для автоматического деплоя через Cloudflare Builds:

- Repository: `egor125552/-archipelago-`
- Production branch: `main`
- Root directory: оставить пустым
- Build command: `npm ci && npm test`
- Deploy command: `npx wrangler deploy`

После подключения каждый коммит в `main` будет запускать проверку и публикацию.
